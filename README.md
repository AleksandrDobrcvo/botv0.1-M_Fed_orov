# Telegram WebApp - Game Profile

Telegram WebApp mini-application with a gaming interface for user profile management.

## Features

### UI Components
- **Header Section**: Game title with 3 stat cards (balance, rating, language selector)
- **Profile Title**: Wooden-style header with decorative elements
- **Missions Section**: Interactive mission card with slider indicators
- **Profile Card**: Main user profile with avatar and statistics grid
- **Bottom Navigation**: 5-button navigation menu with active state
- **Admin Button**: Separate admin panel access

### Design Features
- **Gaming UI Style**: 3D elements, gradients, shadows, rounded corners
- **Smooth Animations**: Hover effects, click animations, fade-in transitions
- **Responsive Design**: Optimized for mobile devices
- **Multi-language Support**: RU/UA/EN language switching
- **Telegram Integration**: Full Telegram WebApp API support

## Language Support

The application supports three languages:
- **RU** (Russian) - Default
- **UA** (Ukrainian) 
- **EN** (English)

Language preference is saved in localStorage and persists across sessions.
